simple_publisher

A simple program to that publishes the current time whenever ENTER is pressed.

Usage:

1 ./simple_publisher [address [port [topic]]]

Where address is the address of the MQTT broker, port it the port number the MQTT broker is running on, and topic is the name of the topic to PUBLISH times to. Note that all these arguments are optional and the defaults are address = "test.mosquitto.org", port = "1883", and topic = "datetime".
